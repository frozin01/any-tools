# Any Tools

## Summary

Any Tools is a tools website that provides various useful tools.

Current tools:
- Word Counter (live words, characters, and characters without spaces)

## Tech Stack
- React

## Getting Started

### 1. Install dependencies

```bash
cd any-tools
npm install
```

### 2. Run development server

```bash
npm run dev
```

Then open the local URL shown in the terminal.

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```
